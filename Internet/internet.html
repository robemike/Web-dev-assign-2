Protocol

A common software standard/language that computers use to communicate over the Internet. Examples include HTTP, FTP, SMTP, TCP/IP.

HTTP

The primary protocol used by the World Wide Web to send information. It is a request-response protocol. HTTP requests are of specific types: GET, POST, HEAD, PUT, DELETE, CONNECT, OPTIONS, or TRACE. Uses port 80 by default.

HTTPS

The secure version of HTTP. Uses port 443.

File Transfer Protocol (FTP)

A protocol that allows files to be easily copied to and from remote sites on the Internet.

Simple Mail Transfer Protocol (STMP)

A protocol within TCP/IP used for sending email messages.

Telnet

An Internet service that allows a user to connect to a remote computer.

Gopher

An early Internet service for information retrieval (predating the Web).

Usenet

An Internet service for discussion groups and message boards.

Email

An Internet service for sending and receiving electronic messages.
